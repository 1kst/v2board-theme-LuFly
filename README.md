# V2Board-theme-LuFly(首页)

## 部署方法

进入v2Board的运行目录：

```shell
cd V2board/public
```

下拉首页文件：

```shell
git clone https://github.com/GZ1903/v2board-theme-LuFly.git .
```

修改Nginx运行的index优先级：

```shell
index index.html index.htm index.php;
```

> 宝塔搭建的，修改配置文件，将index.html放在第一，其他往后移。
